# 日本語ライティング規約

## 丸括弧

- 丸括弧は常に**半角括弧 `( )`** を使う。全角括弧 `（）` は使わない。
- 括弧の**外側に半角スペースを1つ**入れる (開き括弧の前・閉じ括弧の後)。
- 括弧の**内側にはスペースを入れない**。
- 閉じ括弧の直後に句読点 (`。` `、`) が続く場合は、句読点が区切りになるため後続スペースを省く。

### 例

- 正: `Drizzle ORM (drizzle-orm/d1) を利用する`
- 正: `エッジ環境に対応している (WASM 非対応のため)。`
- 誤: `Drizzle ORM（drizzle-orm/d1）を利用する` — 全角括弧、外側スペースなし
- 誤: `Drizzle ORM ( drizzle-orm/d1 ) を利用する` — 内側スペースあり
